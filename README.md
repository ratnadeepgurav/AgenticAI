# CyberWatch — Cybersecurity Monitoring Dashboard

**Demo:** https://cyberwatch2.netlify.app/

A lightweight frontend dashboard for monitoring cybersecurity events and status. Built with a modern Vite-based frontend, the app provides an at-a-glance view of system health, alerts, and basic analytics useful for SOC-style monitoring.

## Features
- Clean, responsive dashboard UI for security metrics
- Widgets for alerts, event summaries, and quick stats
- Searchable/listable items with concise visual indicators
- Mobile-friendly layout for quick checks on the go
- Easy to deploy (Netlify / static hosting)

## Tech stack
- Frontend: Vite + (React / Vue / Svelte) — (project uses Vite for bundling)
- CSS: Modern responsive layout (CSS / Tailwind or plain CSS depending on repo)
- Deployment: Static hosting (Netlify)

> _Note:_ The site is a static frontend app produced by Vite, so it can be hosted on Netlify, GitHub Pages, or any static file host.
